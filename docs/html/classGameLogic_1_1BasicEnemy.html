void GameLogic::BasicEnemy::move ( )
overridevirtual

Function that moves the Basic Enemy to it's next position in the grid and updates movingX and movingY. The Basic Enemy moves from left to right, then lowers a row before moving from right to left. It then lowers a row again before moving from left to right again.
